Advanced Number Guessing Game

Rules

  • Guess the correct number within the given number of digits.
  • Each digit must be chosen from 1-9 (0 is not allowed).
  • A correct digit in the correct place gets a "Correct Place" point.
  • A correct digit in the wrong place gets a "Correct Digits" point.
  • An incorrect digit gets an "Incorrect Digits" point.
  • Score is calculated based on time, attempts, and use of hints.
  • Each level has a different number of digits to guess.
  • Use the "Hint" button to get a clue about the first digit.

Scoring

The score is calculated based on the following criteria:

  • Time taken: Completing the game faster earns more points.
  • Number of attempts: Fewer attempts earn more points.
  • Use of hints: Using hints deducts points.

The final score is determined by:

  • Time Score: Starts with an initial score of 5,000. Deducts 10 points for every second elapsed.
  • Attempts Score: Starts with an initial score of 5,000. Deducts 50 points for each attempt.
  • Hint Penalty: Deducts 500 points if a hint is used.

The score is capped at a minimum value of 0 to prevent negative scores.
